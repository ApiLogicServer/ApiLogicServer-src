Business Background

 

Story: Place Order

As a Business Owner, 

I want to enable Partners to Place Orders via RESTful APIs and Web/Mobile Appis,

So that they can self-serve to reduce their / our Adminstration Costs, and reduce delays.

 

Acceptance Criteria: Supplier Alert

Given a Supplier (such as Pavlov) that has reqistered a Webhook (in Supplier.URL),

When an order is placed containing any of their products,

Then ensure that the system Posts a message to the Supplier in an agreed-upon API format (per SupplierAlert).

 

Technical Approach:

  1. On each Order Details, accrue the webHook URLs, using B2B.putPropertyMap
  2. On Order, process each accrued WebHook URL, by calling B2B.transformToWebHook
